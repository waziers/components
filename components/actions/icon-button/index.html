IconButton

IconButton's are used when you need an icon to trigger an action, such as showing a dialog or deleting an item. It require a label that describes it's action. The label should be descriptive and succinct. If you need an icon for purely decorative purposes use an <Icon /> instead.

;() => {
  const ButtonCollection = styled.div`
    display: grid;
    grid-auto-flow: column;
    grid-gap: ${({ theme }) => theme.space.medium};
    justify-items: center;
    align-items: center;
  `
  return (
    <ButtonCollection>
      <IconButton icon="Plus" label="Add Something Neat" />
      <IconButton icon="Gear" label="Settings" />
      <IconButton icon="Trash" label="Trash It" />
      <IconButton icon="Close" label="Close" />
      <IconButton icon="Favorite" label="Add to Favorites" />
    </ButtonCollection>
  )
}

IconButton Options

Sizes

  • xxsmall | xsmall (default) | small | medium | large
;() => {
  const ButtonCollection = styled.div`
    display: grid;
    grid-auto-flow: column;
    grid-gap: ${({ theme }) => theme.space.medium};
    justify-items: center;
    align-items: center;
  `
  return (
    <ButtonCollection>
      <IconButton icon="Favorite" label="Add to Favorites" size="xxsmall" />
      <IconButton icon="Favorite" label="Add to Favorites" />
      <IconButton icon="Favorite" label="Add to Favorites" size="small" />
      <IconButton icon="Favorite" label="Add to Favorites" size="medium" />
      <IconButton icon="Favorite" label="Add to Favorites" size="large" />
    </ButtonCollection>
  )
}

Shapes

  • round | square (default)

        size="large"
      />
    </ButtonCollection>

}

  • danger | neutral (default) | primary | secondary

Animation

Create the styled rules desired and wrap the IconButton component on it.

;() => {
  const HoverVariantButton = styled(IconButton)`
    transition: transform 0.5s;
    &:hover {
      transform: rotate(45deg);
    }
  `
  return (
    <HoverVariantButton
      label="Close"
      icon="Close"
      size="large"
      outline
      mr="small"
    />
  )
}

IconButton & Tooltip

You can also use your own Tooltip around IconButton (this will effectively remove the "built-in" Tooltip)

<Tooltip content="I'm a little teapot">
  {(eventHandlers, ref) => (
    <IconButton
      {...eventHandlers}
      ref={ref}
      label="Add Something Neat"
      icon="Plus"
      size="xxsmall"
    />
  )}
</Tooltip>
