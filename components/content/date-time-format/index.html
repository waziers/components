DateTimeFormat

The DateTimeFormat component will return a string with a language sensitive representation of the date and time. The specific moment can be specified as children or the default value will be assumed according to the default locale.

Feb 10, 2020, 4:50:11 PM
<Flex justifyContent="space-around" alignItems="center">
  <DateTimeFormat />
</Flex>

DateFormat

Use this component to show only the Date.

Feb 10, 2020
<Flex justifyContent="space-around" alignItems="center">
  <DateFormat />
</Flex>

TimeFormat

Use this component to show only the Time.

4:50:11 PM

Specify locale and Date

Pass the Date desired as children and the locale as prop.

list supported locales

Feb 3, 2011, 5:06:02 AMFeb 10, 2020, 4:50:11 PM
<Flex justifyContent="space-around" alignItems="center">
  <Text>
    <DateTimeFormat>{new Date(Date.UTC(2011, 1, 3, 5, 6, 2))}</DateTimeFormat>
  </Text>
  <Text>
    <DateTimeFormat locale="ar-EG" />
  </Text>
</Flex>

TimeZone

TimeZone needs to be a specific string based on the IANA standard

When adding the timeZone the format will be long - look below for details on formating.

February 10, 2020 at 10:20:11 PM GMT+5:30
<Flex justifyContent="space-around" alignItems="center">
  <Text>
    <DateTimeFormat timeZone="Asia/Kolkata" format="long" />
  </Text>
</Flex>

Format

* medium is the default format if not otherwise specificied.

FORMATOUTPUT
Short2/10/20, 4:50 PM
Medium *Feb 10, 2020, 4:50:11 PM
LongFebruary 10, 2020 at 4:50:11 PM UTC
FullMonday, February 10, 2020 at 4:50:11 PM Coordinated Universal Time
